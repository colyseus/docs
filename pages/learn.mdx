# Learn Multiplayer Game Development with Colyseus

Learn how to build authoritative multiplayer games with Colyseus. Explore demo projects, watch talks, and see how the same game server powers clients across Unity, Godot, Defold, GameMaker, Haxe, Construct 3, and the Web.

## Demo Projects

Each demo shares a single authoritative server with multiple frontend implementations — one per game engine — so you can see exactly how each SDK integrates with Colyseus.
