## Installation

<Tabs items={['Package Manager', 'CDN', 'Download']}>
<Tabs.Tab>
If you're using a build tool (`vite`, `webpack`, `rollup`, or similar):

```sh npm2yarn
npm install --save @colyseus/sdk
```
</Tabs.Tab>

<Tabs.Tab>
Include directly from unpkg. Replace `@x.x.x` with a version compatible with your server:

```html
<script src="https://unpkg.com/@colyseus/sdk@^0.16.0/dist/colyseus.js"></script>
```
</Tabs.Tab>

<Tabs.Tab>
Download the release binaries from [GitHub Releases](https://github.com/colyseus/@colyseus/sdk/releases) and include them in your project:

```html
<script src="colyseus.js"></script>
```
</Tabs.Tab>
</Tabs>

## Initializing the SDK

Create a `Client` instance pointing to your Colyseus server:

```js
import { Client } from "@colyseus/sdk";

const client = new Client("http://localhost:2567");
```

<Callout type="info">
Use `https://` for secure connections in production.
</Callout>

## Quick Example

Here's a minimal example to get you started:

```js
import Colyseus from "@colyseus/sdk";

const client = new Colyseus.Client("http://localhost:2567");

async function main() {
    const room = await client.joinOrCreate("game_room");

    console.log("Joined!", room.sessionId);

    // Send a message to the server
    room.send("move", { direction: "up" });

    // Listen for messages from the server
    room.onMessage("notification", (data) => {
        console.log("Server says:", data);
    });

    // Handle disconnection
    room.onLeave((code) => {
        console.log("Disconnected:", code);
    });
}

main();
```
